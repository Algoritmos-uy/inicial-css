# PROPIEDADES PARA DEFINIR DIMENSIONES EN CSS

-width: Define el ancho de un elemento
-height: Define la altura de un elemento
-max-width: Define el ancho máximo de un elemento
-max-height: Define la altura máxima de un elemento
-min-width: Define el ancho mínimo de un elemento
-min-height: Define la altura mínima de un elemento
-padding: Define el espacio interno de un elemento respecto a su contenido
-margin: Define el espacio externo de un elemento respecto a otros elementos
-font-size: Define el tamaño de la fuente
-line-height: Define la altura de línea
-border-width: Define el grosor del borde
-gap: Define el espacio entre elementos (en flexbox/grid)

## UNIDADES DE MEDIDA EN CSS

=========================================

## UNIDADES ABSOLUTAS

- px (píxeles): Unidad fija, no cambia con el tamaño de la pantalla
- pt (puntos): 1pt = 1/72 de pulgada
- cm (centímetros): Centímetros reales
- mm (milímetros): Milímetros reales
- in (pulgadas): Pulgadas reales

## UNIDADES RELATIVAS

- % (porcentaje): Relativo al elemento padre
- em: Relativo al tamaño de fuente del elemento padre
- rem: Relativo al tamaño de fuente del elemento raíz (html)
- vw (viewport width): 1vw = 1% del ancho de la ventana
- vh (viewport height): 1vh = 1% del alto de la ventana
- vmin: Menor valor entre vw y vh
- vmax: Mayor valor entre vw y vh
- ch: Ancho del carácter "0" de la fuente actual
- ex: Altura de la letra "x" de la fuente actual

## OTRAS UNIDADES

- fr (fracción): Usada en CSS Grid para distribuir espacio
- auto: El navegador calcula automáticamente
